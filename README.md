# <Random-password-generator>

## description

A website with the funtionality to produce a strong random password based on user criteria.

## instalation

N/A

## Usage

To use this website click the generate password button on the bottom of the page. you wll be prompted with some questions about password criteria. select "Ok" for the options needed or hit "cancel" on the ones not needed. you will then be presented with your randomly generated password.
